What Do HTML, CSS, & JavaScript Do?

They are technologies behind what appears when you visit a page o the web.
This is a very surface level explanation of their function.

HTML Function:

Structure: The basic scaffolding of the page, such as he page headings, the text paragraphs, and where the images appear. This is the realm of HTML.

CSS Function:

Style: How the page looks, including fonts, colors, and margins.
This is the function of CSS.

JavaScript Function:

Dynamism: Extras that make the page perform actions such as interacting with the user, "listening" for mouse clicks and key-presses and writing content to the page based on certain conditions.
This is the job of JavaScript.
